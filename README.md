# SMMWE Cloud Level Manager

#### ES

 SMMWE Cloud es una plataforma para compartir niveles de SMMWE creada por YidaozhanYa, gratuita para todos los jugadores de SMMWE. Puedes compartir tus niveles de SMMWE en ella.

SMMWE Cloud Level Manager es el cliente para PC de SMMWE Cloud, puedes utilizarlo para gestionar tus niveles de SMMWE más fácilmente y utilizar todos los servicios de SMMWE Cloud.

#### EN

SMMWE Cloud is a SMMWE level sharing platform created by YidaozhanYa, free for all SMMWE players. You can share your SMMWE levels on it.

SMMWE Cloud Level Manager is the PC client of SMMWE Cloud, you can use it to manage your SMMWE levels more easily and use all the services of SMMWE Cloud.
